# CodeAlpha Data Visualization

## Project Overview

This project analyzes the Sample Superstore dataset using Python and creates professional visualizations to understand business performance.

## Objective

To transform raw sales data into meaningful visual insights that support business decision-making.

## Dataset

Sample Superstore Dataset

The dataset includes:

- Sales
- Profit
- Discount
- Quantity
- Category
- Region
- State
- Order Date

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations

1. Sales by Category
2. Profit by Category
3. Sales Distribution by Region
4. Monthly Sales Trend
5. Top 10 States by Sales
6. Sales vs Profit
7. Discount vs Profit
8. Quantity Distribution
9. Profit Distribution
10. Correlation Heatmap

## Key Insights

- Technology category generated the highest sales.
- Profit varied significantly across categories.
- Some high-sales products produced low profit.
- Higher discounts often reduced profitability.
- Sales showed noticeable variation over time.
- Regional performance differed across the United States.
